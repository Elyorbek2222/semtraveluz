# Урок 6: Ваш первый скрипт SEO-анализа

Время создать что-то реальное. В этом уроке вы создадите скрипт, который анализирует любую веб-страницу на SEO-проблемы.

## Что делает скрипт

По заданному URL скрипт:

- ✅ Загружает HTML страницы через axios
- ✅ Парсит через cheerio
- ✅ Проверяет мета-заголовок (есть? правильная длина?)
- ✅ Проверяет мета-описание (есть? правильная длина?)
- ✅ Считает H1-теги (должен быть ровно 1)
- ✅ Считает H2-теги (должно быть 3+)
- ✅ Считает внутренние ссылки
- ✅ Подсчитывает общее количество слов
- ✅ Оценивает плотность ключевых слов
- ✅ Выводит форматированный отчёт

## Логика проверок

### Meta Title
- ✅ **PASS**: 50-60 символов, содержит главное ключевое слово
- ❌ **FAIL**: < 30 или > 70 символов
- ⚠️ **WARN**: 30-50 или 60-70 символов (не оптимально)

### Meta Description
- ✅ **PASS**: 120-160 символов, содержит CTA
- ❌ **FAIL**: < 80 или > 200 символов
- ⚠️ **WARN**: 80-120 или 160-200 символов (не оптимально)

### H1 Tags
- ✅ **PASS**: ровно 1 H1
- ❌ **FAIL**: 0 H1 или более 1 H1
- **Почему?** Google считает, что на странице должна быть одна главная тема

### H2 Tags
- ✅ **PASS**: 3 или более H2 тегов
- ❌ **FAIL**: менее 3 H2 тегов
- **Почему?** H2 разбивают контент на логические разделы

### Внутренние ссылки
- ✅ **PASS**: 3+ внутренние ссылки
- ⚠️ **WARN**: 1-2 ссылки
- ❌ **FAIL**: 0 ссылок
- **Почему?** Помогают Googlebot обнаруживать контент и распределять авторитет

### Количество слов
- ✅ **PASS**: 800+ слов
- ⚠️ **WARN**: 500-800 слов
- ❌ **FAIL**: < 500 слов
- **Почему?** Глубокий контент ранжируется лучше, чем тонкие статьи

### Плотность ключевых слов
- ✅ **PASS**: 1-2%
- ⚠️ **WARN**: 0.5-1% или 2-2.5%
- ❌ **FAIL**: < 0.5% или > 2.5%
- **Почему?** Google должен видеть релевантность, но без переспама

## Пример кода (для понимания)

```javascript
import axios from 'axios';
import * as cheerio from 'cheerio';

async function analyzeSEO(url) {
  try {
    // 1. Загружаем страницу
    const response = await axios.get(url);
    const html = response.data;
    
    // 2. Парсим HTML
    const $ = cheerio.load(html);
    
    // 3. Извлекаем данные
    const title = $('title').text();
    const description = $('meta[name="description"]').attr('content');
    const h1Count = $('h1').length;
    const h2Count = $('h2').length;
    const internalLinks = $('a[href^="/"]').length;
    const wordCount = $('body').text().split(/\s+/).length;
    
    // 4. Проверяем условия
    const titleOK = title.length >= 50 && title.length <= 60;
    const descOK = description.length >= 120 && description.length <= 160;
    const h1OK = h1Count === 1;
    const h2OK = h2Count >= 3;
    const linksOK = internalLinks >= 3;
    const wordOK = wordCount >= 800;
    
    // 5. Выводим результаты
    console.log(`Meta Title: ${titleOK ? '✅' : '❌'}`);
    console.log(`Meta Description: ${descOK ? '✅' : '❌'}`);
    // ... и так далее
    
  } catch (error) {
    console.error('Ошибка:', error.message);
  }
}

// Запуск
const url = process.argv[2];
if (!url) {
  console.error('Использование: node seo-analyzer.js <URL>');
  process.exit(1);
}

analyzeSEO(url);
```
