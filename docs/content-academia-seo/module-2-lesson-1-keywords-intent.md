# Модуль 2, Урок 1: Ключевые слова и поисковое намерение

Ключевые слова — мост между тем, что люди ищут, и контентом, который вы создаёте. Но не все ключевые слова одинаковы — понимание намерения поиска так же важно.

## 4 типа ключевых слов

### 1. Основные ключевые слова
- **Что:** Главная тема статьи
- **Объём:** 2-4 слова
- **Количество:** Одно на статью
- **Пример:** "SEO автоматизация"
- **Где использовать:** H1, title, первый абзац

### 2. Вторичные ключевые слова
- **Что:** Поддерживающие термины
- **Количество:** 3-5 на статью
- **Пример:** "автоматическая генерация контента"
- **Где использовать:** H2, основной текст
- **Назначение:** Раскрывают аспекты основного ключевого слова

### 3. LSI-ключевые слова
- **Что:** Семантически связанные термины (Latent Semantic Indexing)
- **Количество:** 5-8 на статью
- **Примеры:** 
  - "поисковая оптимизация"
  - "органический трафик"
  - "SEO стратегия"
  - "ранжирование в поиске"
- **Назначение:** Помогают Google понять контекст и тему
- **Почему важны:** Google использует LSI для проверки релевантности

### 4. Длиннохвостые ключевые слова
- **Что:** Конкретные фразы из 4+ слов
- **Объём поиска:** Меньше
- **Конверсия:** Выше (более специфичные)
- **Примеры:**
  - "как автоматизировать генерацию SEO-контента с AI"
  - "система автоматизации SEO для малого бизнеса"
  - "AI инструменты для написания SEO статей"
- **Почему важны:** Низкая конкуренция, высокая конверсия

## Стратегия выбора ключевых слов

```
На одну статью:

1 основное ключевое слово (4-6 слов)
    ↓
3-5 вторичных (поддерживающих)
    ↓
5-8 LSI (семантически связанных)
    ↓
2-3 длиннохвостых (конкретные фразы)
```

## 4 типа поискового намерения

### 1. Информационное (Informational Intent)
- **Что хочет пользователь:** Узнать информацию
- **Запросы:** "Как оптимизировать SEO?", "Что такое LSI?", "Почему SEO важна?"
- **Нужен контент:** Гайды, туториалы, объяснения, статьи
- **Пример результата:** Длинная статья с объяснениями

### 2. Коммерческое (Commercial Intent)
- **Что хочет пользователь:** Исследовать перед покупкой
- **Запросы:** "Лучшие SEO инструменты 2026", "Топ 5 AI для контента", "Сравнение SEO платформ"
- **Нужен контент:** Сравнения, списки "лучших", обзоры, отзывы
- **Пример результата:** Таблица сравнения, рейтинги инструментов

### 3. Транзакционное (Transactional Intent)
- **Что хочет пользователь:** Купить или зарегистрироваться
- **Запросы:** "Купить SEO курс", "Заказать генерацию контента", "Подписка на AI инструмент"
- **Нужен контент:** Лендинги, страницы цен, формы регистрации, checkout
- **Пример результата:** Страница с ценой и кнопкой "Купить"

### 4. Навигационное (Navigational Intent)
- **Что хочет пользователь:** Найти конкретный сайт или страницу
- **Запросы:** "Codearia academy вход", "OpenAI ChatGPT", "Claude Anthropic login"
- **Нужен контент:** Главная страница, логин, навигационные страницы
- **Пример результата:** Прямая ссылка на нужную страницу

## Почему намерение важно для ранжирования

### Принцип Google: Search Intent Alignment
Google сопоставляет результаты поиска с намерением запроса.

**Пример:**
- Запрос: "как создать сайт" (информационное)
- Google показывает: гайды и туториалы
- Google НЕ показывает: страницы с ценами (не матчит намерение)

**Если вы создадите контент неправильного типа, Google не ранжирует его высоко.**

## Карта: Намерение → Тип контента

| Намерение | Тип контента | Примеры |
|-----------|------------|---------|
| **Информационное** | Гайды, туториалы, объяснения | "Как сделать X", "Что такое Y", "Почему Z" |
| **Коммерческое** | Сравнения, списки "лучших", обзоры | "Лучшие X инструменты", "X vs Y", "Топ 5" |
| **Транзакционное** | Лендинги, страницы цен, checkout | Кнопка купить, форма заказа, прайс-лист |
| **Навигационное** | Главная, логин, конкретные страницы | Главная страница, личный кабинет, контакты |

## Практический пример для SEM Travel

### Запрос: "Турция виза 2026"
- **Намерение:** Информационное (пользователь хочет узнать как получить визу)
- **Контент:** Полный гайд по получению визы в Турцию
- **Структура:**
  - H1: "Как получить визу в Турцию в 2026"
  - H2: Требования к визе
  - H2: Документы которые нужны
  - H2: Пошаговый процесс
  - H2: Стоимость и сроки
  - H2: Часто задаваемые вопросы

### Запрос: "Лучшие туры в Дубай"
- **Намерение:** Коммерческое (пользователь исследует перед покупкой)
- **Контент:** Сравнение туров, рейтинги отелей
- **Структура:**
  - H1: "Лучшие туры в Дубай 2026: топ 10"
  - H2: Самые дешёвые туры
  - H2: Люкс туры
  - H2: Семейные туры
  - H2: Таблица сравнения цен

### Запрос: "Забронировать тур в Таиланд"
- **Намерение:** Транзакционное (готов купить)
- **Контент:** Страница бронирования, цены, форма заказа
- **Структура:**
  - Карточки туров с ценами
  - Кнопки "Забронировать"
  - Форма контактных данных

## Как наша система это делает

**Совет:** Наша система автоматически классифицирует намерение для каждого ключевого слова.

1. **Анализ:** AI анализирует ключевое слово
2. **Классификация:** Определяет тип намерения (информ./комм./транзакц./навиг.)
3. **Генерация:** Создаёт контент нужного типа
4. **Валидация:** Проверяет что контент матчит намерению

## Главный вывод

**Ключевое слово без понимания намерения — как карта без знания пункта назначения.**

Вы можете оптимизировать идеально, но если контент не соответствует тому, что ищет пользователь, Google не ранжирует его.

### Правило: Намерение > Ключевые слова

Сначала поймите что хочет пользователь, потом создавайте контент.
