# Модуль 2, Урок 3: Анализ ключевых слов конкурентов

Лучшая стратегия ключевых слов не рождается в вакууме. Анализируя, по каким запросам ранжируются конкуренты, вы находите пробелы — темы, которые они упускают.

## Что показывает анализ конкурентов

### 1. Их фокус
- Какие темы приоритетны?
- На каких ключевых словах они ранжируются?
- Какие топики их основная ниша?

### 2. Пробелы
- **Что они упускают?**
- Какие подтемы не покрыты?
- Какие ключевые слова недоиспользованы?

### 3. Структура контента
- Как организованы их статьи?
- Какие паттерны заголовков?
- Какой объём контента оптимален?

### 4. Углы дифференциации
- Как предоставить **больше ценности**?
- Какой уникальный угол добавить?
- Как выделиться среди конкурентов?

## Скрапинг мета-тегов конкурентов

С помощью **cheerio** извлекаем ключевые SEO-данные:

```javascript
import axios from 'axios';
import * as cheerio from 'cheerio';

async function scrapeCompetitor(url) {
  const response = await axios.get(url);
  const $ = cheerio.load(response.data);
  
  return {
    title: $('title').text(),
    description: $('meta[name="description"]').attr('content'),
    h1: $('h1').map((_, el) => $(el).text()).get(),
    h2: $('h2').map((_, el) => $(el).text()).get(),
    h3: $('h3').map((_, el) => $(el).text()).get(),
    wordCount: $('body').text().split(/\s+/).length,
    images: $('img').length,
    links: $('a').length,
  };
}
```

## Поиск пробелов в ключевых словах

### 1. Пропущенные подтемы
- **Что:** Конкуренты не покрывают определённые аспекты
- **Пример:** Все пишут о "как работает SEO", но никто не объясняет "E-E-A-T сигналы"
- **Ваша стратегия:** Напишите специализированную статью на пропущенную подтему

### 2. Поверхностное покрытие
- **Что:** Один абзац, где можно написать статью
- **Пример:** Конкурент уделяет "интеграциям" один абзац
- **Ваша стратегия:** Напишите 1000-словную статью на эту тему

### 3. Устаревший контент
- **Что:** Статья 2023-2024 года, вы можете дать свежие данные
- **Пример:** "SEO тренды 2024" — написана год назад
- **Ваша стратегия:** Обновите и переопубликуйте как "SEO тренды 2026"

### 4. Несоответствие намерению
- **Что:** Написали туториал, когда пользователь хочет обзор
- **Пример:** Статья "как установить" когда нужна "таблица сравнения"
- **Ваша стратегия:** Проанализируйте намерение и создайте правильный контент

## Стратегии дифференциации

### 1. Глубже (Depth)
```
Если конкурент пишет 500 слов
→ Вы пишите 1000 с примерами

Если конкурент дает 3 совета
→ Вы даете 10 + кейсы
```

### 2. Визуально (Visuals)
```
Если конкурент только текст
→ Вы добавляете:
  - Код (примеры)
  - Диаграммы (процессы)
  - Скриншоты (пошаговые)
  - Таблицы (сравнения)
```

### 3. Актуальнее (Freshness)
```
Конкурент: "SEO тренды 2024"
Вы: "SEO тренды 2026 + новые алгоритмы"

Конкурент: Старые цены
Вы: Актуальные цены с 2026 году
```

### 4. Уникальный угол (Unique Angle)
```
Конкурент: "Как работает SEO"
Вы: "Как мы с 0 до 12K посетителей через SEO — реальный кейс"

Конкурент: Теория
Вы: Реальные кейсы и практика
```

## Практический пример для SEM Travel

### Запрос: "Тур в Турцию"

**Что делают конкуренты:**
- Показывают общие описания мест
- Перечисляют отели
- Не упоминают визу

**Пробелы:**
- ❌ Виза требования не покрыты
- ❌ Бюджетные варианты упущены
- ❌ Сезонность не рассмотрена

**Ваша стратегия:**
- ✅ Включите визу гайд
- ✅ Добавьте бюджетные туры
- ✅ Объясните "лучшее время для поездки"
