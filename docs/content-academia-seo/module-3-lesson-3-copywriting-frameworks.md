# Модуль 3, Урок 3: Копирайтинг-фреймворки: AIDA, PAS, 4U

Отличный SEO-контент — это не только ключевые слова, но и вовлечение читателя. Профессиональные копирайтеры используют проверенные фреймворки. Мы встраиваем их прямо в системный промпт AI.

## AIDA — для основных разделов

Attention, Interest, Desire, Action — используйте для каждого раздела статьи:

**Attention** — Начните с яркого утверждения или статистики. 
> «90% блог-постов получают 0 трафика из Google.»

**Interest** — Раскройте детали. 
> «Причина? Контент пишется для людей, но не оптимизируется для поисковиков.»

**Desire** — Покажите пользу. 
> «Представьте: одна статья приносит 500 посетителей ежемесячно — автоматически.»

**Action** — Дайте конкретный шаг. 
> «Вот как структурировать статью для максимального SEO-эффекта.»

## PAS — для введений

Problem, Agitate, Solution — самый эффективный фреймворк для введений:

**Problem** — Назовите боль. 
> «Вы часами пишете статьи, а Google их игнорирует.»

**Agitate** — Усугубите. 
> «А конкуренты ранжируются на 1-й странице с AI-контентом за 5 минут.»

**Solution** — Дайте выход. 
> «В этом гайде вы построите ту же систему — шаг за шагом.»

## 4U — для заголовков

Каждый заголовок должен пройти тест 4U:

**Useful** — Обещает ли заголовок пользу?
**Urgent** — Есть ли причина читать СЕЙЧАС?
**Unique** — Выделяется ли среди аналогов?
**Ultra-specific** — Есть ли цифры, сроки, конкретика?

### Сравните:

```
❌ Плохо: «SEO-советы для блога»
   (не срочно, не конкретно)

✅ Хорошо: «7 SEO-правок, которые удвоили наш трафик за 30 дней»
   (полезно, срочно, конкретно)
```

## Когда какой фреймворк

```
Введение → PAS
Основные разделы → AIDA
Заголовки и H2 → 4U
Заключение → Bridge (сейчас → цель → путь)
```

## Совет для AI

AI следует фреймворкам гораздо лучше, когда вы явно называете их в промпте. 

❌ Не пиши вовлекающе
✅ А используй PAS для введения

## Главный вывод

**Фреймворки превращают средний контент в убедительный. AI знает эти паттерны — вам нужно лишь указать, какой где использовать.**
