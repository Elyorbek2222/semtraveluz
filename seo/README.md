# SEM Travel.uz — SEO Avtomatizatsiya

## Papka tuzilmasi

```
/seo
├── /schema              — Schema.org markup (JSON-LD)
├── /sitemap            — Sitemap generation
├── /config             — SEO configuration
├── /utils              — Utility functions
├── /audit              — Audit reports va analytics
└── /templates          — SEO template patterns
```

## Komponenta ro'yxati

| Fayl | Tavsif |
|------|--------|
| `config/seo.config.ts` | SEO global configuration |
| `schema/tour-schema.ts` | Tour product schema |
| `schema/org-schema.ts` | Organization schema |
| `sitemap/sitemap-generator.ts` | XML sitemap generation |
| `utils/meta-tags.ts` | Meta tags utility |
| `utils/structured-data.ts` | Structured data generator |
